ACADEMIC GRADING SYSTEM

In this project, we will be building an Academic Grading system. In our grading system, you will be able to put in a number grade, and the system will spit out a letter grade, for example 80 = B. This project will we be done using a combination of HTML, CSS, and our Programming Foundations in JavaScript. Non of your applications functionality should be done in the console. Requirements
  1. HTML Input to allow the user to input grades(numbers)
  2. The input should only allow a number type to be inputted.
  3. The input should only allow number 0 - 100.
  4. Using JavaScript and DOM Manipulation, display the letter grade on the page.
  5. Should allow a single or multiple inputs at once.
  6. If multiple inputs are entered, the systems should return each individual letter grade as well as an average.
